# Kiurate

This is a social media clone made for Bath Spa University

## Stack

- [Next.js](https://nextjs.org)
- [Clerk](https://clerk.com)
- [Prisma](https://prisma.io)
- [Tailwind CSS](https://tailwindcss.com)
- [tRPC](https://trpc.io)
- [MongoDB](mongodb.com/)

## How to Run (Developer Mode)

You need to set the env variables to run, please see .env.local for the required env variables.

- `npm install`
- `npx prisma generate`
- `npm run dev`
